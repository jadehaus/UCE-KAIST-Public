우리가 직접 만들어가는
프로그램, 매화수.

매화수는 과목 수강 설계를 도와주는 프로그램입니다.
앞으로는 카이스트 학생이라면 누구나 참여할 수 있는 오픈소스 프로젝트로 확대할 계획입니다. (혼자 만들고 유지하는건 버거우니까요.)

매화수는 어떤 프로그램인가요?

매화수의 시작은 엑셀

매화수는 본래 졸업 요건 확인을 돕는 엑셀 파일이었습니다. 엑셀은 사용자가 쉽게 함수를 수정하고 정보를 변경할 수 있다는 장점이 있지만, 업데이트 및 추가 기능 등을 제공하기 어렵다는 단점이 있었습니다. 초기에 배포했던 엑셀 파일은 여기에서 확인할 수 있습니다.

새롭게 만들어진 웹배전, 매화수

새롭게 웹사이트로 만들어진 매화수는 언제 어디서든 쉽고 간편하게 졸업 요건을 확인할 수 있도록 제작되었습니다. 나아가, 사용자의 익명 데이터들을 기반으로 AI 등을 활용해 스마트한 과목 추천 시스템을 구현할 예정입니다.

함께 만들어가는 프로젝트, 매화수

매화수에는 아직 많은 기능이 없습니다. 하지만, 여러분이 도와주신다면 (제발) 다채로운 매화수를 함께 만들어갈 수 있을 것입니다. 베타 테스팅 기간이 끝난 이후에 매화수를 함께 개발할 분을 공개적으로 모집할 예정입니다.

안녕하세요, 매화수 개발자인
수리과학과 학사과정 김민우입니다.
매화수 베타 버전 웹사이트에 접속해주신
여러분께 진심으로 감사드립니다.

엑셀 버전의 매화수를 접해보신 분들은 아시겠지만, 매화수는 졸업 요건을 계산해주고 과목 수강 설계를 도와주는 프로그램입니다. 2학년 이상의 학생은 학교 학사시스템의 졸업관리시스템을 사용할 수 있지만, 학교의 졸업관리시스템은 "앞으로 내가 무슨 과목을 더 들어야 졸업할 수 있지?"와 같은 질문에 대한 대답을 효과적으로 알려주진 못했습니다. 그리고 갓 입학한 새내기들은 아예 졸업관리시스템을 사용할 수도 없지요.

정확한 졸업 요건을 계산해준다는 점에서 졸업을 앞둔 고학년에게는 학교의 시스템이 훨씬 중요하겠지만, 저학년 학생들에게는 앞으로의 수강 설계가 훨씬 중요하므로 학교의 시스템이 지원해주지 못한 무언가를 만들고 싶다는 생각이 있었습니다.

이번에 새롭게 웹사이트 매화수는, 이러한 과목 설계 기능에 초점을 두고 만들어졌습니다. 비록 아직은 간단한 졸업 요건 계산 등의 기능만이 기초적으로 구현되어 있지만, 앞으로 사용자가 늘어난다면 축적된 데이터를 바탕으로 '학업 계획 설계'에 충실한 매화수를 만들어갈 수 있을 것입니다.

쉽고 빠른 과목 추가하기(학사시스템)

*해당 방법은 학사시스템의 로그인을 필요로 합니다.

  1. 학사시스템(https://cais.kaist.ac.kr/grade)에 로그인하여
    성적 > 성적조회 클릭.
  2. Chrome의 경우, 화면에서 마우스 우클릭 후
    가장 아래 보이는 검사 (페이지 소스보기 아래) 클릭.
  3. 검사창 가장 위 console 창 클릭 후,
    아래 코드를 콘솔 창에 복사 붙여넣기.

    4. let el = document.getElementById("gradeContent");
    let rows = el.querySelectorAll("tr");
    let crows = Array.from(rows).filter(r => {return (r.id.length==4 || r.id==='0');});
    let ccrows = Array.from(crows).filter(cr => {return ((cr.querySelector("td:nth-of-type(11)").innerText != 'R') && (cr.querySelector("td:nth-of-type(11)").innerText != 'W'));});
    let courses = ccrows.map(cr => {
    return {"year": parseInt(cr.id), "semester": cr.getAttribute("name"),
    "course_id": cr.querySelector("td:nth-of-type(3)").innerText,
    "code": cr.querySelector("td:nth-of-type(4)").innerText,
    "name": cr.querySelector("td:nth-of-type(7)").innerText};});
    console.log(JSON.stringify(courses));

  4. 콘솔창 아래 출력된 결과값을 아래 텍스트 상자에 입력하면 현재까지 수강한 과목들이(재수강 및 드랍 과목 제외) 성공적으로 반영됩니다.
  *일부 개별연구와 기이수 과목들의 경우 반영되지 않을 수 있습니다.
